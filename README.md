# #management
계정, 사이트 그룹 관리 프로젝트

# #목적? 목표?
1. 스프링 학습을 최우선으로 할 것
2. 회사에서 진행 중인 것과 관련된 내용으로 진행할 것
3. 개발 규칙에 맞추어 진행할 것.

# #요구 사항
- spring security를 사용하여 로그인/로그아웃 구현
- 각 사이트는 그룹별로 접근 권한이 다름
- 각 메뉴는 그룹별로 접근 권한이 다름
- 메뉴는 각 사이트별로 각각 구성됨
- 메뉴는 계층구조를 가짐
